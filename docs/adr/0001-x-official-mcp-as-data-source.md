# 数据源采用 X 官方 MCP

需要从 X 获取某领域的热点推文及其互动数据。选定 X 官方 MCP（`https://api.x.com/mcp`，只读 Bearer token 直连、按量付费），因为它是官方支持、数据完整（全档案搜索 + 互动指标 + 作者信息），且成本随用随付、无固定月费门槛。

## Considered Options

- **浏览器自动化抓取**（agent-reach / Playwright）— 零 API 成本但脆弱（反爬、DOM 变动）、有账号风险，不适合无人值守。
- **其他AI厂商的实时检索 API** — 语义检索契合，但多引入一个厂商依赖且互动数据不如官方完整。
- **第三方数据 API** — 便宜但灰色地带。

## Consequences

数据访问层仍做成可替换的抽象——后期计划支持其他获取手段与非 X 内容源，MCP 只是 MVP 的默认实现。
